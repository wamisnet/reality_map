# Character assets

公開ページとオペレーターコンソールで使用するキャラクター画像。

## 置き換え方法

ここに置いてある SVG はプレースホルダーです。実画像で置き換える場合は同じファイル名で **同じディレクトリ** に配置してください。拡張子を変えたい (PNG / WebP) 場合は `src/data/character.ts` のパス定義も合わせて編集します。

| ファイル | 使われる場面 |
|---|---|
| `default.svg` | TOP ページのヒーロー |
| `pointing.svg` | 結果リスト (参加者名を指す) |
| `rank-s.svg` | 個別結果ページ・S ランク。`/ops` の lock 演出にも使用 |
| `rank-a.svg` | 個別結果ページ・A ランク |
| `rank-b.svg` | 個別結果ページ・B ランク |
| `rank-c.svg` | 個別結果ページ・C ランク |

## 推奨サイズ

縦長 (例: 1:1.4 程度) で背景透過。表示サイズは概ね 240×340 を基準にしていますが、SVG / 高解像度ラスタなら拡縮できるので素材自体は大きめでOK。
